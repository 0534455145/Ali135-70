# Ali135-70 — User Profile Layout

A clean, accessible user profile page with improved layout for better visibility.

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/0534455145/Ali135-70.git
   cd Ali135-70
   ```

2. **Open the profile page**

   No build step required. Simply open `index.html` in your browser:

   ```bash
   open index.html   # macOS
   # or
   xdg-open index.html   # Linux
   # or double-click the file in your file explorer
   ```

3. **Customise the profile**

   - Edit `index.html` to update the name, title, bio, skills, and activity items.
   - Edit `style.css` to adjust colours, spacing, or typography.

## Project Structure

```
Ali135-70/
├── index.html   # User profile page
├── style.css    # Styles for the profile layout
└── README.md    # Project documentation
```

## Features

- **Prominent profile header** — avatar, name, title, and location at a glance
- **Online status indicator** — green badge on the avatar
- **Stats bar** — projects, followers, and following counts
- **Skills tags** — colour-coded pill badges
- **Recent activity feed** — chronological list of user actions
- **Responsive design** — adapts gracefully to narrow screens
- **Accessible markup** — semantic HTML, ARIA labels, and sufficient colour contrast
